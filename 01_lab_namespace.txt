2. How is autoloading initiated?

~Zend/workspaces/DefaultWorkspace/orderapp/public/index.php sets BASE to ~Zend/workspaces/DefaultWorkspace/orderapp/ and then sends the anonymous function below to spl_autoload_register():

    function ($class) {
        $file = str_replace('\\', '/', $class) . '.php';
        require BASE . '/src/' . $file;
    }
 
With this autoloader, a class X\Y\Z for example, is parsed into
$file = X/Y/Z.php
and then loaded by:
require ~Zend/workspaces/DefaultWorkspace/orderapp/src/X/Y/Z.php
 
The classes in the vendor folder like Guzzle and Monolog do not appear that they would be autoloaded because the composer autoloader is commented out:

//require '../vendor/autoload.php';
